# HackToolOS
Entorno personalizado orientado a la ciberseguridad basado en Debian 11, al estilo de Kali o Parrot. Es un entorno muy minimalista con la intención de que pueda funcionar en equipos con configuraciones de hardware muy sencillas.
# Descarga
Se puede descargar en formato OVA desde el siguiente enlace de Google drive.

https://drive.google.com/drive/folders/1S_DqYWMf7bqKR_g9H2O8EPPOriLWtf9p
# Ejecución
Es compatible con la mayoría de software de virtualización (VMWare, Virtual Box, etc.)

Algunas de las aplicaciones instaladas requieren escalar permisos con “sudo” para ello se pueden utilizar las siguientes credenciales.

Usuario: hacktool

Contraseña: hacktool
